# Sales Analysis
The portfolio is aimed at creating dashboards that help monitoring overall performance of a retailing firm.

## 2. Detailed analysis 
* SellChannelPerformance: Which sales channel brings the most customers?
* Countries: Comparison of sales performance among different countries.
* ManagerPerformance: Comparison of managers' performance.
* HomeProduct, ScreenProduct, Videogames, Books: Sell status of each product category.
